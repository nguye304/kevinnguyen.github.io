Finesse Nais

Mobile App developed with Xamarin Forms

I developed this mobile app for my Mom's nail salon. It uses Xamarin in order to be cross platform. It allows users to access information about the employees, prices, hours, and location. It also allows users to make reservations with employees. This app gets its data from a mySQL database. I used AWS to set up the database and developed an API for the app to access it. The API was developed using Node.js.

  • Date: January 2019
  • Client: Finesse Nails
  • Category: Mobile App

Anime Discovery App

Mobile App developed using Kotlin & Android Studio.

I had a problem of trying to find new anime to watch so I used Kotlin make an android app. It uses a free unofficial myanimelist API and acts alot like tinder where you swipe right to add the anime to your "to-watch" list and swipe left to ignore. If the user clicks on the image, they can access more information about the anime. It also allows users to access their lists which are stored on a mysql database on AWS.

  • Date: February 2019
  • Client: Kevin Nguyen
  • Category: Mobile app

Student Exchange

Website developed using HTML, CSS, JS, and PHP

This was a school project developed for a Web Development Course. It is an amazon/craigslist clone that allows students to sell and trade their things. It uses HTML, CSS, Javascript and Bootstrap for the front end. The backend uses PHP and SQLi. This website allows users to create personal accounts, browse items and access their shopping cart. I was responsible for the development of the profile page and shopping cart. The profile page will change according to who is logged in using SESSIONS and will have the items the user is selling. The shopping cart shows what is currently in the user's shopping cart and will calculate the total.

  • Date: December 2018
  • Client: CSUSM
  • Category: Web Site
